    ⓘ Playlist import only — no login needed to add songs

    To link a Spotify track when adding a song, just paste the Spotify URL in the add-song form — no account needed.

    Playlist bulk import requires Spotify login, which needs an HTTPS redirect URI. The easiest way:

    Option A — ngrok (free):

    npx serve . -p 8080 npx ngrok http 8080

    Copy the https://xxxx.ngrok.io URL as your Redirect URI.

    Option B — GitHub Pages (permanent HTTPS):

    Push the 3 files to a GitHub repo → enable Pages → use https://username.github.io/repo/index.html.

    Register the chosen HTTPS URL in Spotify Developer Dashboard → your app → Edit → Redirect URIs.
